# 🧪 Calculadora Estequiométrica Inteligente (TCC)

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://www.python.org/)
[![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-orange.svg)]()

Este projeto é um software desenvolvido como Trabalho de Conclusão de Curso (TCC) no Ensino Médio. Trata-se de uma ferramenta robusta para automação de cálculos químicos, focada em precisão e facilidade de uso para estudantes e profissionais da área.

## 🚀 Objetivo
Transformar a lógica complexa da estequiometria em algoritmos eficientes, aplicando conceitos de **Engenharia de Software** para resolver problemas reais de química, como balanceamento de equações e identificação de reagentes limitantes.

## 🛠️ Funcionalidades Planejadas
- [x] **Parser de Fórmulas:** Interpretação de strings químicas (ex: `H2SO4`, `C6H12O6`) usando Expressões Regulares (Regex).
- [ ] **Balanceamento Automático:** Resolução de sistemas lineares para balanceamento de equações.
- [ ] **Cálculo de Massa Molar:** Integração com banco de dados de massas atômicas.
- [ ] **Análise Reacional:** Identificação de reagente limitante, excesso, rendimento e pureza.
- [ ] **Módulo de Gases:** Cálculos baseados na lei dos gases ideais ($PV = nRT$).

## 💻 Tecnologias Utilizadas
- **Linguagem:** Python
- **Ambiente de Desenvolvimento:** VS Code (via GitHub Codespaces / Xbox Series S)
- **Lógica Matemática:** Álgebra Linear e Regex

## 📂 Estrutura do Projeto (Em construção)
- `/core`: Lógica principal do motor de cálculo.
- `/data`: Arquivos JSON com dados da Tabela Periódica.
- `/tests`: Scripts de testes unitários para validar os cálculos.
